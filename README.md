# WMS — склад автомобилей

Простая система управления автомобильным складом на FastAPI. Интерфейс позволяет добавлять, искать, перемещать и удалять автомобили, а также показывает занятость склада по зонам.

## Запуск

Установите зависимости:

```bash
pip install fastapi uvicorn pydantic
```

Запустите API из корня проекта:

```bash
uvicorn api:app --app-dir api --reload
```

API будет доступен по адресу `http://127.0.0.1:8000`, интерактивная документация — `http://127.0.0.1:8000/docs`.

Для запуска интерфейса откройте папку `frontend` через локальный HTTP-сервер, например:

```bash
py -m http.server 5500 --directory frontend
```

После этого откройте `http://127.0.0.1:5500`.

## API Endpoints

### Получить конфигурацию склада

```http
GET /api/warehouse
```

Возвращает информацию о складе и список всех его ячеек.

### Получить дашборд

```http
GET /api/dashboard
```

Пример ответа:

```json
{
  "totalCars": 1,
  "totalCells": 58,
  "occupiedCells": 1,
  "freeCells": 57,
  "zones": [
    {
      "zone": "A",
      "total": 6,
      "occupied": 1,
      "free": 5
    }
  ]
}
```

### Добавить автомобиль

```http
POST /api/car
Content-Type: application/json

{
  "vin": "WVWZZZ1JZXW000001",
  "model": "Volkswagen Golf",
  "cellId": "A-1-2",
  "arrivalTime": "2026-07-14T06:09:07.487Z"
}
```

Поля `cellId` и `arrivalTime` необязательны. Если `cellId` не указан, сервер выберет первую свободную ячейку. Если `arrivalTime` не указан, будет использовано текущее время UTC.

Возможные ошибки:

- `409` — автомобиль с таким VIN уже существует;
- `409` — выбранная ячейка занята или на складе нет свободных мест.

### Найти автомобиль по VIN

```http
GET /api/car/{vin}
```

| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| `vin` | `string` | **Обязательный.** VIN автомобиля |

Возвращает `404`, если автомобиль не найден.

### Получить все автомобили

```http
GET /api/cars
```

Возвращает массив автомобилей. Если склад пуст, возвращается пустой массив.

### Найти автомобили по модели

```http
GET /api/cars/{model}
```

| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| `model` | `string` | **Обязательный.** Название модели |

Пример:

```http
GET /api/cars/Volkswagen%20Golf
```

Возвращает `404`, если автомобили указанной модели не найдены.

### Переместить автомобиль

```http
PATCH /api/car/{vin}
Content-Type: application/json

{
  "cellId": "B-1-2"
}
```

Освобождает прежнюю ячейку и занимает новую. Возвращает `404`, если автомобиль или ячейка не найдены, и `409`, если новая ячейка занята.

### Удалить автомобиль

```http
DELETE /api/car/{vin}
```

При удалении автомобиля его ячейка освобождается. Возвращает `404`, если VIN не найден.

### Получить первый прибывший автомобиль модели (FIFO)

```http
GET /api/fifo/{model}
```

Возвращает автомобиль указанной модели с самым ранним `arrivalTime` или ошибку `404`, если таких автомобилей нет.

## Структура проекта

```text
api/
  api.py             FastAPI-маршруты
  models.py          модели данных Pydantic
  MemoryStorage.py   хранилище автомобилей в памяти
  topology.json      конфигурация зон и ячеек склада
frontend/
  index.html         разметка интерфейса
  App.js             запросы к API и обновление дашборда
  style.css          стили интерфейса
```

Данные автомобилей хранятся в оперативной памяти и сбрасываются при перезапуске API. Топология склада загружается из `api/topology.json`.
